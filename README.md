# Smart Data Analyzer (Python)

## Description
A Python tool that automatically analyzes CSV or Excel files and generates a clear summary report with row counts, column counts, duplicate detection, and missing value analysis.

## Features
- Supports CSV and Excel files
- Detects duplicate rows
- Identifies missing values per column
- Generates a client-ready summary report

## Tech Stack
- Python
- Pandas
- OpenPyXL

## How to Run
```bash
pip install -r requirements.txt
python analyzer.py your_file.xlsx

Output
A text report named data_summary.txt containing dataset insights.

Use Case
Helpful for freelancers, analysts, and small businesses to quickly understand raw data files.
